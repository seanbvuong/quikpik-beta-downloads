# quikpik beta downloads

This repository contains packaged beta builds of quikpik, a local-first browser extension for saving, finding, and pasting reaction pics on websites you allow.

## Install beta 8

1. Download and unzip `quikpik-v0.1.0-beta.8.zip`.
2. Open your Chromium browser's extensions page and enable **Developer mode**: `chrome://extensions` in Chrome, `edge://extensions` in Edge, or `brave://extensions` in Brave.
3. Choose **Load unpacked** and select the unzipped folder.
4. Open quikpik from the toolbar, add reaction pics and keywords, then allow a website in Settings.
5. Refresh any website tabs that were already open.

Saved pics stay in your local browser profile. This beta supports up to 50 pics, with a 20 MB limit per pic.

This repository intentionally contains packaged builds only, not the source code.
